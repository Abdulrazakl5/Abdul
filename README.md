 VeloPay Solutions Cybersecurity Capstone Project

Overview
This project develops and implements a cybersecurity improvement plan for VeloPay Solutions, a boutique FinTech payment processing company experiencing rapid growth and increasing cybersecurity risks.

The project aligns with:
- NIST Cybersecurity Framework (CSF)
- PCI-DSS compliance requirements
- Zero Trust security principles
- Least Privilege access control



 Business Challenges
- Weak network segmentation
- No centralized SIEM logging
- Excessive local administrator privileges
- Lack of formal incident response plan
- Hybrid workforce security gaps
- Insecure guest Wi-Fi access
- Limited visibility into AWS S3 access logs


 Security Goals
1. Achieve PCI-DSS audit readiness
2. Enforce MFA for all remote access
3. Reduce Mean Time to Detect (MTTD) to under 15 minutes
4. Segment guest Wi-Fi from production systems
5. Implement centralized logging and monitoring
6. Enforce Principle of Least Privilege (PoLP)


 Technologies Used
- Microsoft 365 Business Premium
- Microsoft Intune
- Microsoft Entra ID
- Ubiquiti UniFi Firewall
- AWS S3 Logging
- Windows Server 2019
- QuickBooks Desktop
- Syslog/SIEM Integration



Security Architecture
The project implements:
- VLAN segmentation
- Zero Trust access policies
- Conditional Access
- Endpoint encryption with BitLocker
- SIEM log aggregation
- Secure VPN remote access
- Automated alerting
